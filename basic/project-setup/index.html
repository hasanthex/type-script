Short Hand Details of TypeScript

# Content Details 
TypeScript setup & config
1 Convert .ts file to .js file 
 => tsc typescript_file_name.ts
2 Create tsconfig.json file 
 => tsc --init
3 Convert .ts file to .js file after setup tsconfig.json file 
 => tsc
4 Convert .ts file to .js file automatically. 
 => tsc -w 
TypeScript Basic
5 Strict Types Variables 
let player = "Mr. Doo";
console.log(player);

======== output ========
Mr. Doo

let player = "Mr. Doo";
player = 30;
console.log(player);

======== output ========
Type 'number' is not assignable to type 'string'.
6 Strict Types Method 
function multiply(a, b){
  return a*b
}
console.log(multiply('10',10))

======== output ========
Parameter 'b' implicitly has an 'any' type.

function multiply(a : number, b : number){
  return a*b
}
console.log(multiply('10',10))

======== output ========
Argument of type 'string' is not assignable to parameter of type 'number'.
7 Strict Types Array 
let country = ["Bangladesh", "India"];
country.push(51)
console.log(country)

======== output ========
 Argument of type 'number' is not assignable to parameter of type 'string'.

let country = ["Bangladesh", "India"];
country.push("Australia")
console.log(country)

======== output ========
["Bangladesh", "India", "Australia"]

let country = ["Bangladesh", "India", 1];
country.push("Australia")
country.push(30)
console.log(country)

======== output ========
['Bangladesh', 'India', 1, 'Australia', 30]
